# 🚀 DHIRAJ OS

A futuristic cyberpunk-inspired personal portfolio website showcasing my journey, skills, projects, certifications, achievements, and professional experience in Computer Science, Cybersecurity, Machine Learning, and Web Development.

![Portfolio Banner](assets/images/banner.png)

---
Live Demo :-
https://dhirajmarathe2007.github.io/dhiraj-os-portfolio/

## 🌟 Overview

DHIRAJ OS is a modern single-page portfolio designed with a space-tech aesthetic and glassmorphism UI. The website presents my technical profile in an engaging and interactive format.

### Key Highlights

* ✨ Cyberpunk & Space-inspired design
* 🎯 Fully responsive layout
* 🚀 Interactive animations and effects
* 💎 Glassmorphism UI
* 📱 Mobile-friendly navigation
* 🌌 Particle background effects
* 📊 Skills and achievements showcase
* 🏆 Certifications section
* 💼 Experience timeline
* 📬 Contact form
